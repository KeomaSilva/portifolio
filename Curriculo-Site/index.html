LEK'S Quiz

O projeto LEK'S Quiz foi inicialmente pensado para uma tarefa da disciplina de Introdução às tecnologias. O grupo imaginou uma plataforma de quiz, com o intuíto de auxiliar os professores do ensino médio com atividade mais interativas com a tecnologia, e para os alunos que querem estudar ter outra ferramenta para a assimilação do conteúdo. Com isso foi criado uma HomePage para o site que seria para hospedar a plataforma de quiz.
